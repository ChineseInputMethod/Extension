这个词库是我多年前收集整理的，已经忘记了原始出处，是多个词库的并集。
我忘记了是如何协调不同词库词频的，大概只记得大于11的是真实词频。
等于11的是我人工为某个高质量词库设置的词频。
等于10的是我为高质量词库并集中词频为1的，乘以10的结果，也就是说大于10的词频除以10，就是原词频。
（为协调一般质量和低质量词库的词频，进行了一些调整，并人工设置了多音词的词频）
小于10的是一般质量词库收录的统计，并非真实词频。
（等于10和11的词频，均可以还原为1，不存在真实词频或收录统计等于10或11的情况）
（有可能当初为高质量词库加权乘以100，一般质量乘以的10，具体记不清了）

`large_pinyin.txt`是[mozillazg](https://github.com/mozillazg/phrase-pinyin-data)中的收录的词库，我对其进行形式上的审阅，并将词频设置为10。

`core_other.txt`是我的原词库与`large_pinyin.txt`交集的补集，进行过人工审核。

`expand_other.txt`是我的原词库与`large_pinyin.txt`交集的补集，只进行了程序审核。

`repeat.txt`是上述词库中的多音词。
